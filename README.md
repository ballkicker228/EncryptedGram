# EncryptedGram
Скрипт для шифрованной переписки через telegram.

![Alt demo](demo.png)

1. Клонируй репозиторий:

    ```
    git clone https://github.com/ballkicker228/EncryptedGram
    ```

2. Открой директорию с программой:

    ```
    cd EncryptedGram
    ```

3. Установи зависимости:

    ```
    poetry install
    ```

4. Запусти программу:

    ```
    poetry run main
    ```
